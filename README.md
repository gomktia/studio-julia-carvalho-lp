# Estudio Júlia Carvalho - Landing Page

Landing page desenvolvida para o Estudio Júlia Carvalho, focada em cursos de estética e automaquiagem.

## Tecnologias Utilizadas

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como rodar o projeto localmente

```sh
# 1. Instale as dependências
npm i

# 2. Inicie o servidor de desenvolvimento
npm run dev
```

## Implantação

O projeto pode ser implantado em qualquer plataforma de hospedagem estática como Vercel ou Netlify.
